# Competitive Programming Resources

## Knowledge

- [Book: _Competitive Programmer's Handbook_ — Antti Laaksonen](resources/books/book.pdf) (local PDF, 296 pages)
  THE primary text for this mission. Written specifically for ICPC-style prep. Use for: every topic — each lesson maps to a chapter. Ch 1–7 (basics, complexity, sorting, data structures, complete search, greedy, DP) are the 10-day core; ch 11–13 (graphs) second; ch 21 (number theory) third.
- [CSES Problem Set](https://cses.fi/problemset/)
  The official companion problem set to the handbook, by the same author. Use for: drilling each chapter's technique immediately after reading it. Start with "Introductory Problems" and "Sorting and Searching".
- [USACO Guide](https://usaco.guide/)
  Free, curated, structured curriculum with explanations and problem lists. Bronze/Silver sections match current level. Use for: a second explanation when the handbook is too terse.
- [Codeforces Problemset](https://codeforces.com/problemset)
  Filter by rating (900–1400) and tag. Use for: volume practice calibrated to the zone of proximal development.
- [ICPC Asia Pacific past problems](https://icpc.global/worldfinals/problems)
  Past regional problem sets. Use for: the day-9 full 5-hour simulation and understanding the real difficulty profile.
- [AtCoder Beginner Contests](https://atcoder.jp/contests/?ratedType=1)
  Clean, well-calibrated problems A–D. Use for: timed mini-sets and virtual contests on partial days.
