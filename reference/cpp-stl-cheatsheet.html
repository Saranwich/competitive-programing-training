C++ Template & STL Essentials

The template you paste every time, and the handful of STL tools that solve most easy problems. Never web-search "which header is iota in" again.

The template

Start every file with this. <bits/stdc++.h> pulls in every standard header at once — so vector, iota (from <numeric>), sort, set, map are all already there. You never include individual headers again.

#include <bits/stdc++.h>
using namespace std;
using ll = long long;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    // solve here

    return 0;
}

The two ios lines make cin/cout fast (matters when reading 10⁵+ numbers). using ll = long long; so you type ll when the constraint says you'll overflow int — see complexity sheet.

numeric — the ones you forgot

GoalCodeHeader (all in bits/stdc++)
fill a, b, a+1 … (a=0)iota(v.begin(), v.end(), 0);<numeric>
sum of a vectoraccumulate(v.begin(), v.end(), 0LL);<numeric>
Overflow trap. accumulate(v.begin(), v.end(), 0) sums as int and overflows. The 0LL seed forces long long. This is the 1083 lesson baked into one keystroke.
